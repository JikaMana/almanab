# Al Manāb — Astro Site

A small static site showcasing Al Manāb's dual focus: premium Islamic apparel and software/tech services. Built with Astro, GSAP animations, and Lucide icons.

## Quick start

Requirements:

- Node.js (current LTS recommended)
- npm

Install dependencies:

```sh
npm install

npm run dev

npm run build

npm run preview

/ (project root)
├── public/                  # static public assets
│   └── robots.txt
├── src/
│   ├── assets/              # images and static assets
│   │   └── almanab-apparels.png
│   ├── components/
│   │   └── Home.astro       # main page UI
│   ├── layouts/
│   │   └── Layout.astro     # layout + JSON-LD
│   ├── pages/
│   │   └── index.astro      # page entry
│   ├── scripts/
│   │   └── animations.ts    # GSAP + ScrollTrigger + icons
│   └── styles/
│       └── global.css
├── [astro.config.mjs](http://_vscodecontentref_/0)
└── [package.json](http://_vscodecontentref_/1)
```
